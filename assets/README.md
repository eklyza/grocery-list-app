# Assets

Add the following image files to this directory:

- `icon.png` - App icon (1024x1024 px recommended)
- `splash.png` - Splash screen image (1242x2436 px recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

You can generate these from a single logo using tools like:
- [Expo Icon Generator](https://github.com/expo/expo/tree/main/packages/%40expo/image-utils)
- [App Icon Generator](https://appicon.co/)
